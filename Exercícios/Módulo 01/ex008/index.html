Principais Formatações

Negrito / Destaque

Neste parágrafo existe um termo em negrito usando a tag <b> (não semântica).

Neste parágrafo existe um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Neste parágrafo existe um termo em itálico usando a tag <i> (não semântica).

Neste parágrafo existe um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido, mas não deve ser considerado, com a tag <del>.

Texto inserido

Também é possível criar um texto como inserido, para dar ênfase em que ele foi adicionado depois. Isso com a tag <ins>.

Texto sobrescrito

Para escrever coisas como x3 + 4. Usa-se a tag <sup>.

Texto subescrito

Para escrever coisas como H2O, com a tag <sub>.
